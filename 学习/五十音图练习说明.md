# 🈷️ 五十音图练习模块使用说明

## 📚 模块概述

五十音图练习模块是专为日语初学者设计的交互式学习工具，帮助你快速掌握日语的基础——平假名（ひらがな）和片假名（カタカナ）。

## ✨ 功能特点

### 1. **双模式学习**
- 🇯🇵 **平假名模式**：学习和练习 50 个平假名字符
- 🔤 **片假名模式**：学习和练习 50 个片假名字符
- 可随时切换，灵活学习

### 2. **三种练习方式**

#### 📊 五十音图表
- 完整展示所有假名及其读音
- 按行分组显示（あ行、か行、さ行...）
- 包含清音、浊音、半浊音
- 悬停查看读音，方便记忆

#### 📝 认读练习（选择题模式）
- 看假名，选择正确的罗马音
- 或看罗马音，选择正确的假名
- 四个选项，实时反馈
- 自动记录得分和正确率

#### ⌨️ 打字练习（输入模式）
- 看假名，手动输入罗马音
- 训练肌肉记忆和拼写
- 支持回车键快速提交
- 更接近实际使用场景

### 3. **智能题目生成**
- 随机抽取假名，避免死记硬背
- 干扰选项科学设计，增加辨识难度
- 可选择特定行进行集中练习
- 题目类型随机变化，保持新鲜感

### 4. **实时进度追踪**
- 显示当前题目数量
- 实时统计正确率
- 即时反馈答题结果
- 鼓励性提示语

## 🎯 学习建议

### 第1周：平假名清音（46个）
1. 每天学习 2-3 行（10-15 个假名）
2. 先看图表熟悉字形和读音
3. 用认读练习巩固（每行至少 20 题）
4. 用打字练习强化记忆（每行至少 10 题）

**推荐顺序：**
- 第 1-2 天：あ行、か行、さ行
- 第 3-4 天：た行、な行、は行
- 第 5-6 天：ま行、や行、ら行、わ行
- 第 7 天：复习所有

### 第2周：片假名清音（46个）
使用相同的学习方法，学习片假名。由于已经掌握了读音，这周会更轻松。

### 第3周：浊音和半浊音（25个）
- 浊音：が行、ざ行、だ行、ば行
- 半浊音：ぱ行
- 理解浊音符号（゛）和半浊音符号（゜）的作用

### 第4周：综合复习
- 混合练习平假名和片假名
- 增加练习速度和准确率
- 目标：3秒内识别任意假名

## 📈 评估标准

### 初级（学习第1-2周）
- 认读正确率 ≥ 70%
- 能在 5 秒内识别常见假名

### 中级（学习第3-4周）
- 认读正确率 ≥ 85%
- 能在 3 秒内识别任意假名
- 打字练习正确率 ≥ 80%

### 高级（学习1个月后）
- 认读正确率 ≥ 95%
- 能在 2 秒内识别任意假名
- 打字练习正确率 ≥ 95%
- 不再需要思考，形成条件反射

## 💡 学习技巧

### 1. **形象记忆法**
- あ：像一个人张开嘴说"啊"
- き：像一把钥匙（key）
- す：像一个人在睡觉（sleep）
- 找到每个假名的形状特点

### 2. **对比记忆法**
注意容易混淆的假名：
- **平假名**：あ/お、は/ほ、ね/わ/れ、ぬ/め
- **片假名**：シ/ツ、ソ/ン、ウ/ワ、ク/ケ

### 3. **书写练习**
- 虽然这个模块主要是认读，但建议配合纸笔练习
- 边写边读，记忆更牢固
- 注意笔顺，养成正确书写习惯

### 4. **每日复习**
- 每天至少练习 15-20 分钟
- 复习前一天学过的内容
- 使用"错题集"功能重点复习错误的假名

### 5. **实际应用**
- 尝试用假名写自己的名字
- 阅读简单的日语单词
- 听日语歌曲时尝试识别假名

## ❓ 常见问题

**Q: 应该先学平假名还是片假名？**
A: 建议先学平假名。平假名使用频率更高，掌握后再学片假名会更容易。

**Q: 多久能完全掌握五十音图？**
A: 因人而异，一般 2-4 周可以基本掌握。关键是每天坚持练习。

**Q: 为什么有些假名看起来很像？**
A: 这是正常的。通过大量练习，你会逐渐发现它们的细微差别。建议使用"对比记忆法"。

**Q: 打字练习时应该输入什么？**
A: 输入罗马音（如：あ 输入 a，き 输入 ki）。这是日语输入法的基础。

**Q: 可以跳过某些行吗？**
A: 不建议。五十音图是整体系统，建议按顺序全部学习。
