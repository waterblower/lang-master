# 🚀 五十音图练习 - 快速上手指南

## 📍 访问方式

### 方法一：通过导航菜单
1. 打开网站首页
2. 点击顶部导航栏的 **"🈷️ 五十音"** 按钮
3. 进入五十音图练习页面

### 方法二：直接访问
在浏览器中输入：`http://localhost:8000/kana`

---

## 🎯 3分钟快速开始

### 第1分钟：选择假名类型

进入页面后，你会看到两个大按钮：

```
┌─────────────┐  ┌─────────────┐
│     あ      │  │     ア      │
│  平假名      │  │  片假名      │
└─────────────┘  └─────────────┘
```

**建议**：初学者先选择 **平假名**（左边的按钮）

---

### 第2分钟：查看五十音图

选择假名类型后，向下滚动，你会看到完整的五十音图表：

```
あ行: あ い う え お
      a  i  u  e  o

か行: か き く け こ
      ka ki ku ke ko

さ行: さ し す せ そ
      sa shi su se so
      
... (以此类推)
```

**建议**：花2分钟浏览所有假名，熟悉它们的样子

---

### 第3分钟：开始练习

选择一个练习模式：

#### 📝 认读练习（推荐新手）
- 看假名，选择正确的罗马音
- 四选一，有提示
- 适合刚接触假名的学习者

#### ⌨️ 打字练习（进阶）
- 看假名，输入罗马音
- 需要记住拼写
- 适合有一定基础的学习者

---

## 💡 学习小贴士

### 🔰 完全零基础？

**第1天**：
1. 只看图表，不做练习
2. 重点看 **あ行**（a, i, u, e, o）
3. 对比平假名和片假名的区别

**第2-3天**：
1. 开始做认读练习
2. 先练习 **あ行** 和 **か行**
3. 每天练习20题

**第4-7天**：
1. 继续学习新的行
2. 每天复习前面学过的
3. 逐渐增加到每天50题

---

### 📊 如何看懂得分？

练习时，页面顶部会显示：

```
题目: 15    得分: 12 / 15
```

这表示：
- 你已经做了 15 道题
- 答对了 12 道
- 正确率 = 80%

**目标**：
- 初期：正确率 ≥ 70%
- 中期：正确率 ≥ 85%
- 熟练：正确率 ≥ 95%

## 🎮 界面操作指南

### 认读练习模式

```
┌──────────────────────────┐
│  这个假名的读音是？        │
│                          │
│         あ               │  ← 题目
│                          │
│  ┌────┐ ┌────┐           │
│  │ a  │ │ ka │           │  ← 四个选项
│  └────┘ └────┘           │
│  ┌────┐ ┌────┐           │     点击选择
│  │ sa │ │ ta │           │
│  └────┘ └────┘           │
└──────────────────────────┘
```

**操作步骤**：
1. 看题目中的假名
2. 点击你认为正确的罗马音
3. 查看反馈（绿色=正确，红色=错误）
4. 点击"下一题"继续

---

### 打字练习模式

```
┌──────────────────────────┐
│  输入这个假名的罗马音      │
│                          │
│         あ               │  ← 题目
│                          │
│  ┌──────────────────┐    │
│  │ 输入罗马音...     │    │  ← 输入框
│  └──────────────────┘    │
│                          │
│      [提交答案]           │  ← 按钮（或按回车键）
└──────────────────────────┘
```

**操作步骤**：
1. 看题目中的假名
2. 在输入框中输入罗马音（如：`a`）
3. 按回车键或点击"提交答案"
4. 查看反馈
5. 点击"下一题"继续

**提示**：
- 不区分大小写（`A` 和 `a` 都可以）
- 输入完按回车更快
- 适合已经记住读音，想加强记忆的学习者

## 📖 常见问题

### Q1: 应该先学平假名还是片假名？
**A**: 先学平假名。平假名使用频率更高，而且学会了平假名，片假名会更容易学。

### Q2: 每天应该练习多久？
**A**: 建议每天15-30分钟，坚持比时长更重要。

### Q3: 多久能学完？
**A**: 
- 平假名：1-2周
- 片假名：1-2周
- 浊音/半浊音：1周
- 总共：3-5周（因人而异）

### Q4: 为什么有些假名很像？
**A**: 这是正常的！举例：
- 平假名：`ね`、`わ`、`れ` 很像
- 片假名：`シ`、`ツ` 很像；`ソ`、`ン` 很像

**解决办法**：多练习，注意细节差异

### Q5: 可以跳过某些行吗？
**A**: 不建议。五十音图是一个系统，每个假名都很重要。

### Q6: 练习时可以作弊吗？
**A**: 可以随时返回图表查看，学习不需要为难自己。但建议：
- 先尝试凭记忆作答
- 实在不确定再查看
- 记录下不熟悉的假名，重点复习

## 🎯 学习路径推荐

### 📅 4周学习计划

**第1周：平假名清音（あ-わ行）**
- Day 1-2: あ行、か行、さ行
- Day 3-4: た行、な行、は行
- Day 5-6: ま行、や行、ら行、わ行
- Day 7: 复习全部

**第2周：片假名清音（ア-ワ行）**
- Day 8-10: ア-タ行
- Day 11-13: ナ-ワ行
- Day 14: 复习全部

**第3周：浊音和半浊音**
- Day 15-17: 浊音（が、ざ、だ、ば行）
- Day 18-19: 半浊音（ぱ行）
- Day 20-21: 混合练习

**第4周：综合复习**
- Day 22-28: 混合练习所有假名，提高速度和准确率

## 🏆 进阶建议

当你能达到以下标准时，说明你已经掌握了五十音图：

✅ **认读速度**：看到假名后，2秒内说出读音
✅ **准确率**：认读练习正确率 ≥ 95%
✅ **打字速度**：打字练习正确率 ≥ 90%
✅ **双向掌握**：既能看假名说读音，也能听读音写假名
✅ **混合识别**：平假名和片假名混在一起也能识别
